# Lead CRM 🚀

## 📌 Overview
This is a Full Stack Lead Management System built using:
- React (Frontend)
- Node.js + Express (Backend)
- PostgreSQL (Database)

## ✨ Features
- Add new leads
- View all leads
- Update lead status (Interested / Not Interested / Converted)
- Delete leads
- Search and filter leads
- Dashboard with lead statistics
- Form validation
- Clean UI with animations

## 🛠️ Tech Stack
- Frontend: React, Axios
- Backend: Node.js, Express
- Database: PostgreSQL (Neon)

## ▶️ Run Project

### Backend
cd backend  
npm install  
node src/server.js  

### Frontend
cd frontend  
npm install  
npm start  

## 🔗 API Endpoints

### Add Lead
POST /api/leads

### Get All Leads
GET /api/leads

### Update Lead
PUT /api/leads/:id

### Delete Lead
DELETE /api/leads/:id
